### 🚀 Final Year Project — College Info Chatbot

> **Padmashree College Chatbot** — A hybrid NLP chatbot combining Rule-Based logic + Multinomial Naive Bayes + TF-IDF

<div align="center">

| Feature | Detail |
|:---|:---|
| 🏗️ Architecture | 3-Layer Hybrid (Rule-Based → MNB → Confidence Fallback) |
| 🎯 Intent Categories | 90 intents |
| 📦 Training Data | ~45,000 augmented samples |
| 🌐 Language Support | English + Romanized Nepali |
| 🛠️ Stack | Python · scikit-learn · NLTK · Flask · TF-IDF · joblib |

</div>
